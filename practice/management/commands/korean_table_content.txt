한국어 교육 및 학습을 위한 전체 문법 목차입니다. 한글 기초부터 고급 한국어능력시험(TOPIK) 문법까지 빠짐없이 총 86개의 강좌로 통합하여 구성했습니다.

## 1단계: 한글 입문 및 기초 문법 (초급 1)

1. 한글 창제 원리와 자음, 모음
2. 한글 발음 규칙과 받침 (연음, 격음화 등)
3. 인사말, 기본 표현 및 자기소개
4. 명사 + **입니다 / 입니까?** (서술어와 의문어)
5. 조사 1: **은/는** (주제), **이/가** (주어)
6. 조사 2: **을/를** (목적어), **의** (소유)
7. 조사 3: **에, 에서** (장소와 시간)
8. 동사/형용사 + **아/어요** (비격식 존댓말 현재형)
9. 동사/형용사 + **ㅂ니다/습니다** (격식 존댓말 현재형)
10. 동사/형용사 + **았/었어요** (과거형)
11. 부정 표현 1: **안** + 동사/형용사, 동사/형용사 + **지 않다**
12. 부정 표현 2: **못** + 동사, 동사 + **지 못하다** (능력 부족)
13. 한국어 숫자 (고유어, 한자어)와 단위 명사
14. 시간, 날짜, 요일 표현 (시, 분, 년, 월, 일)
15. 의문사 (누구, 무엇, 어디, 언제, 왜, 어떻게)

## 2단계: 기본 의사소통 문법 (초급 2)

16. 불규칙 용언 1 ('ㅂ', 'ㄷ', '으' 불규칙)
17. 동사 + **(으)ㄹ 거예요** (미래 및 추측)
18. 동사 + **고 싶다** (희망 표현)
19. 동사 + **(으)세요 / 지 마세요** (명령과 금지)
20. 동사 + **(으)ㄹ 수 있다/없다** (가능과 불가능)
21. 동사 + **아/어 주다** (요청 및 도움)
22. 연결 어미 1: 동사/형용사 + **고** (나열 및 순서)
23. 연결 어미 2: 동사/형용사 + **지만** (대조)
24. 연결 어미 3: 동사/형용사 + **아/어서** (이유 및 시간적 순서)
25. 동사/형용사 + **(으)면** (조건과 가정)
26. 비교 표현: 명사 + **보다 (더/덜)**
27. 동사 + **기 전에 / (으)ㄴ 후에** (시간적 선후 관계)
28. 동사 + **(으)면서** (동시 동작)
29. 동사 + **(으)려고 하다** (의도와 계획)
30. 동사 + **아/어 보다** (경험과 시도)

## 3단계: 중급 도약 (중급 1)

31. 관형사형 어미 1: 동사 + **는** (현재)
32. 관형사형 어미 2: 동사 + **(으)ㄴ** (과거), **(으)ㄹ** (미래)
33. 관형사형 어미 3: 형용사 + **(으)ㄴ**
34. 명사화 표현: 동사 + **는 것**, 동사 + **기**
35. 불규칙 용언 2 ('르', 'ㅅ', 'ㅎ' 불규칙)
36. 연결 어미 4: 동사/형용사 + **(으)니까** (주관적 이유와 발견)
37. 동사/형용사 + **기 때문에**, 명사 + **때문에** (객관적 원인)
38. 동사 + **아/어야 하다/되다** (의무와 필요)
39. 동사/형용사 + **(으)ㄴ/는/(으)ㄹ 것 같다** (추측)
40. 동사 + **(으)ㄹ 줄 알다/모르다** (방법과 능력)
41. 동사 + **기로 하다** (결정과 약속)
42. 동사/형용사 + **잖아요** (확인 및 강조)
43. 피동 표현 (**-이/히/리/기-**, **아/어지다**)
44. 사동 표현 (**-이/히/리/기/우/구/추-**, **게 하다**)
45. 동사 + **아/어 버리다** (행동의 완료와 감정적 결과)

## 4단계: 중고급 심화 (중급 2 ~ 고급 1)

46. 간접 화법 1: 평서문 (**-다고 하다**), 의문문 (**-냐고 하다**)
47. 간접 화법 2: 명령문 (**-라고 하다**), 청유문 (**-자고 하다**)
48. 동사 + **(으)ㄹ 뻔하다** (과거의 위기 모면)
49. 동사 + **(으)ㄹ 테니까** (화자의 의지 및 추측에 따른 이유)
50. 동사/형용사 + **(으)ㄹ수록** (정도의 심화)
51. 명사 + **(이)나 다름없다** / 동사 + **(으)ㄴ/는 셈이다** (결과적 동일)
52. 동사/형용사 + **(으)ㄴ/는 반면에** (상반된 사실)
53. 동사/형용사 + **(으)ㄹ 뿐만 아니라** (점층과 추가)
54. 동사/형용사 + **(으)ㄴ/는 데다가** (상황의 가중)
55. 동사 + **다가 보면** (행동의 지속과 새로운 결과 발견)

## 5단계: 고급 완성 (고급 2)

56. 동사/형용사 + **(으)ㅁ으로써** (수단과 방법의 명사화)
57. 동사/형용사 + **(으)ㄹ 지경이다** (극단적 상태나 한계)
58. 동사/형용사 + **(으)ㄹ 리가 없다 / (으)ㄹ 턱이 없다** (강한 의심과 부정)
59. 명사 + **에 달려 있다** (조건에 따른 결과)
60. 동사 + **(으)려던 참이다** (막 하려던 의도와 우연의 일치)

## 6단계: 고급 뉘앙스와 심화 표현

61. 부정적 원인과 핑계: 동사 + **는 바람에**, **(으)ㄴ/는 탓에**, **느라고**
62. 후회와 아쉬움: 동사 + **(으)ㄹ걸 (그랬다)**, **았/었어야 했다**
63. 목적과 의도의 심화: 동사 + **(으)ㄹ 겸**, **고자**
64. 당연함과 필연성: 동사/형용사 + **기 마련이다**, **(으)ㄴ/는 법이다**
65. 상태의 지속과 유지: 동사 + **아/어 놓다**, **아/어 두다**
66. 강한 추측과 가능성: 동사/형용사 + **(으)ㄹ지도 모르다**, **기 십상이다**

## 7단계: 시간과 동시 동작의 확장

67. 즉각적인 연속: 동사 + **자마자** / 동사 + **기가 무섭게**
68. 기회와 도중: 동사 + **는 길에** / **는 김에**
69. 행동의 완료 후 새로운 사실: 동사 + **고 나서** / **(으)ㄴ 채로**

## 8단계: 조건, 가정, 그리고 양보

70. 과거에 대한 반대 가정: 동사/형용사 + **았/었더라면**
71. 부정적 결과의 경고: 동사/형용사 + **다가는**
72. 노력의 소용없음: 동사/형용사 + **아/어 봤자**
73. 강한 양보와 가정: 동사/형용사 + **(으)ㄹ지라도** / **더라도** / **(으)ㅁ에도 불구하고**

## 9단계: 정도와 한계

74. 행동이나 상태의 정도: 동사/형용사 + **(으)ㄹ 정도로** / **(으)ㄹ 만큼**
75. 극단적인 상태 강조: 형용사 + **기 짝이 없다**
76. 단지 ~일 뿐이다: 명사 + **에 불과하다** / 동사/형용사 + **(으)ㄹ 따름이다** / **(으)ㄹ 뿐이다**

## 10단계: 선택과 무관함

77. 선택과 무관함: 동사/형용사 + **든지 든지** / **건 건**
78. 더 나은 대안 선택: 동사 + **(느)니 차라리**
79. 행동의 무의미함: 동사 + **(으)나 마나**

## 11단계: 인용의 확장과 감정

80. 들은 사실의 확인: 동사/형용사 + **다면서요** / **냐면서요**
81. 놀라움과 감탄: 동사/형용사 + **다니** / **라니**
82. 당연한 추측과 짐작: 동사/형용사 + **(으)려니 하다**
83. 원망이나 탓: 명사 + **(이)랍시고** / 동사 + **(으)ㄴ/는답시고**

## 12단계: 고급 문장 수사법

84. 근거와 이유의 격식체: 동사/형용사 + **(으)로 말미암아** / **(으)로 인해**
85. 당연한 사실의 반문: 동사/형용사 + **거늘**
86. 극단적 양보와 포기: 동사/형용사 + **기로서니**
